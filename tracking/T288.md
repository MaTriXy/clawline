## Product Truth
Web typing indicator cancel parity follows native T217 source context:

- Tapping the rendered typing indicator opens a compact cancel confirmation bubble anchored to the typing bubble.
- `Cmd-.` opens the same confirmation.
- `Esc` or outside tap dismisses.
- `Return` confirms while the confirmation is open.
- Confirming uses typed `cancel_current_run`; it never sends `/stop` as chat text.

Provider seam tickets are not dependencies for this Web ticket unless a live capability mismatch is independently observed.

## Implementation Notes
Web now opens a compact cancel prompt from the rendered typing indicator or `Cmd-.` / `Ctrl-.` when session status or typing state indicates cancellable assistant work. The confirmation calls `/api/session-control` with `action: "cancel_current_run"`.

The cancel prompt is positioned from the rendered typing bubble's viewport rect so the outside-dismiss backdrop does not block the red Cancel action.
