# Clawline Design System

This is the canonical location for Clawline's visual design system and chat UI demonstrations.

## Files

- **design-system.html** — Design system specification (colors, typography, spacing, component styles)
- **demo.html** — Interactive demonstration of the organic flow chat UI
- **img/** — Screenshots and reference images
- **experiments/** — Historical theme explorations (not in active use)

## Sync

This directory syncs weekly (Sunday 2am PST) to `~/src/clawline/design-system/` on eezo's git repo.

**Direction:** shared-workspace (canonical) → repo (synced copy)
